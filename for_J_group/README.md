## 開発方針
C言語には現在時刻を表示したりするtime関数シリーズが存在します  
それをブレッドボードで使用できるかは定かではありませんが  
まずはtime関数を試してみましょう  
時刻データを拾ってきてからの操作ははＧチームのものと  
ほぼ同じなのでまあお互い頑張りましょう
